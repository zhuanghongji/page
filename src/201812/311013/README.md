# React 实战进阶：20 使用 Jest，Enzyme 等工具进行单元测试


## React 让单元测试变得更容易

1. React 应用很少需要访问浏览器 API
2. 虚拟 DOM 可以在 NodeJS 环境运行和测试
3. Redux 隔离了状态管理，纯数据层单元测试


## 单元测试涉及的工具

* Jest：Facebook 开源的 JS 单元测试框架
* JS DOM：浏览器环境的 NodeJS 模拟
* Enzyme：组件渲染和测试
* nock：模拟 HTTP 请求
* sinon：函数模拟和调用跟踪
* istanbul：单元测试覆盖率
